# 跨会话总结技能
用户想要总结 {type} 为 {target} 的最近 {count} 条聊天记录。

请按照以下步骤执行：
1. 调用工具 `get_history`，传入参数：session_type="{type}", session_id="{target}", count={count}
2. 工具会返回最近的消息列表
3. 根据返回的消息内容，总结这段时间的主要话题和关键点
4. 总结控制在40-120字以内，用自然流畅的语气
5. 如果获取消息失败，请礼貌告知用户“无法获取该会话的消息”

注意：
- 总结时尽量提炼核心观点，不要逐条复述
- 如果消息中有明显的话题转变，可以分段总结
- 最终回复要让人感觉你确实看过聊天记录
